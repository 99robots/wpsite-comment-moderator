Add a new user role, Comment Moderator, that allows a user to only manage comments.

== Installation ==

1. Upload `wpsite-comment-moderator` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to plugins page to see instructions for shortcode and php template tags
